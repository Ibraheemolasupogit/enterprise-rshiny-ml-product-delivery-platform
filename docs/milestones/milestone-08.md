# Milestone 8 - SAS Viya Integration

Status: Externally blocked

Milestone 8 was intended to integrate the candidate model workflow with genuine SAS Viya capabilities: authenticated access, hands-on experiment or model registration evidence, governed endpoint evidence, screenshots or logs from the real platform, and a clear comparison between local and SAS-managed lifecycle controls.

Genuine SAS Viya access is not available in this environment. The local filesystem registry implemented in Milestone 7 is not being represented as SAS Viya, and no fake screenshots, endpoint evidence, model registration exports, approvals or experiments were created.

Evidence required before completion:

- Authenticated SAS Viya workspace access.
- Real model or experiment registration evidence.
- Real endpoint or serving evidence where permitted.
- Governance and approval artefacts generated by the platform.
- Reproducible run notes that distinguish SAS evidence from local fallback evidence.

Decision: continue with the local open-source product path until genuine SAS Viya access is provided.
