# Interview Evidence Index

Status vocabulary:

- `implemented_and_tested`: implemented in source and covered by local tests.
- `live_validated`: exercised against a local live service or explicit live integration path.
- `offline_mock_validated`: implemented with mocked external transport or offline-compatible provider tests.
- `documented_extension`: documented future path with no current implementation claim.
- `requires_external_environment`: requires an available external platform, credentials or network route.
