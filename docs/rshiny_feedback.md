# R-Shiny Feedback

The feedback page collects product feedback only:

- Output understandability.
- Explanation usefulness.
- Usability.
- Governance-status clarity.
- Optional product comment.

It does not ask whether a prediction was clinically correct. It does not request names, email addresses, patient identifiers, operational case details or clinical outcomes.

Local demonstration feedback is written to `data/monitoring/shiny_feedback.jsonl`, which is ignored. Committed evidence records the feedback contract rather than live user comments.
