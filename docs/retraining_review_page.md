# R-Shiny Retraining Review

The R-Shiny Retraining Review page displays committed retraining evidence only. It shows eligibility, dataset counts, champion identity, challenger metrics, comparison deltas, promotion gates and recommendation boundaries.

The page cannot run retraining, register a challenger, approve, activate, deploy or roll back a model. Persistent copy states that the page displays evidence only and cannot mutate lifecycle state.

The recommendation section always surfaces automatic action, approval, activation and deployment status. For Milestone 12 these remain `none`, `not_granted`, `inactive` and `not_performed`.
