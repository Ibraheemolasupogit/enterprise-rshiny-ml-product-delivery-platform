# ADR 0039: Validation Test Degradation Remains Visible

Status: Accepted

Context: Locked test evidence is weaker than validation evidence in several metrics.

Decision: The R-Shiny performance page explicitly displays test specificity 0.4 and balanced accuracy near 0.561.

Consequences: Stakeholders see limitations before any governance decision.

Alternatives considered: Hiding weak metrics or only showing recommendation status was rejected as misleading.
