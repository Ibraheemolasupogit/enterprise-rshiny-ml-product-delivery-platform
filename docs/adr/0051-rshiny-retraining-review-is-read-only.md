# ADR 0051: R-Shiny Retraining Review Is Read Only

## Status

Accepted.

## Context

Operational users need visibility of retraining evidence without gaining model-lifecycle controls.

## Decision

The Retraining Review page reads committed evidence and provides no buttons for retraining, registration, approval, activation, deployment or rollback.

## Consequences

The user interface supports scrutiny and traceability without becoming a governance bypass.
