# R-Shiny Model Governance

The Model Governance page is read-only. It displays registry identity, registry state, approval status, activation status, governance recommendation, review flags, safe audit events, and model-card limitations.

The current state remains registered, approval pending, activation inactive, and governance recommendation defer. Automatic approval is disabled and a human decision is required.

The page does not provide approve, reject, activate, retire, rollback, register, deploy, or promotion controls. It does not alter registry files or model state.
