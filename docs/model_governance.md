# Model Governance

Milestone 7 reviews the registered XGBoost candidate using validation evidence, locked test evidence, calibration, threshold behaviour, fairness, explainability, feature compatibility, reproducibility, and known runtime dependencies.

The governance review surfaces validation-to-test degradation, weak locked-test specificity, small synthetic sample size, suppressed fairness groups, and synthetic-only limitations. The committed demonstration recommendation is `defer`; human accountability remains required before any approval.

Approval is never automatic. Conditional approval requires explicit conditions and an actor identifier.
